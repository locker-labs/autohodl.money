# Features / Why Use

- **Automatic round-ups:** No new habits; savings happen every time you spend.  
- **Aave yield:** Spare change goes straight into Aave to start earning.  
- **Spendable Yield Tokens (SYTs):** Wrapped yield tokens with better spend UX. Send sUSDC, recipients get regular USDC while you keep earning.  
- **Future-proof automation:** Planned migration to ERC-7715 for native, trustless permissions once the MetaMask Delegation Toolkit supports Linea.  
- **Roadmap expansion:** Additional yield protocols and mUSD support as soon as MetaMask Card integration is available.
